# Automated-Web-Testing-Project

This project is a test automation framework built to validate the functionality of the website:
https://automationexercise.com/

Project Goals
- Build a scalable UI test automation framework using the Page Object Model (POM) design pattern
- Derive user stories from application functionality
- Create high-quality test cases and BDD scenarios
- Execute and report on test results
- Practice collaborative development using GitHub
- Implement CI/CD using GitHub Actions
- Perform exploratory testing and defect reporting

## Stack
- Language: Java
- Test Framework: Selenium and Serenity
- BDD: Cucumber
- Build Tool: Maven 
  
## Setup Instructions

### Prerequisites
- Java (JDK 21 recommended)
- Maven
- IntelliJ IDEA (or any IDE)

## What Was Tested

### Core User Flows
- User Registration
- Product Search
- Checkout Process

### Test Types
- Automated UI tests (Selenium + Serenity)
- BDD scenarios (Cucumber)
- Exploratory testing
